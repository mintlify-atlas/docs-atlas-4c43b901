---
title: "Download and Install PineconeMC for Windows, macOS, Linux"
description: "Download and install PineconeMC on Windows, macOS, or Linux. Get stable releases from GitHub or try the latest development builds via nightly.link."
---

PineconeMC is available for Windows, macOS, and Linux. Stable releases are published on the [GitHub Releases](https://github.com/ElyPrismLauncher/Launcher/releases/latest) page. Choose the package for your platform below.

<Tabs>
  <Tab title="Windows">
    1. Go to the [Releases](https://github.com/ElyPrismLauncher/Launcher/releases/latest) page.
    2. Download the `.exe` installer for your architecture (x86\_64 for most systems).
    3. Run the installer and follow the on-screen prompts.
    4. Launch **PineconeMC** from the Start menu or desktop shortcut.
  </Tab>
  <Tab title="macOS">
    1. Go to the [Releases](https://github.com/ElyPrismLauncher/Launcher/releases/latest) page.
    2. Download the `.dmg` disk image.
    3. Open the `.dmg` file and drag **PineconeMC** into your **Applications** folder.
    4. Launch PineconeMC from Launchpad or Spotlight.

    <Note>
      On first launch macOS may show a security warning because the app is not notarized. Open **System Settings → Privacy & Security** and click **Open Anyway** to proceed.
    </Note>
  </Tab>
  <Tab title="Linux">
    Several package formats are available.

    **AppImage**

    1. Download the `.AppImage` file from the [Releases](https://github.com/ElyPrismLauncher/Launcher/releases/latest) page.
    2. Make it executable and run it:

    ```bash
    chmod +x PineconeMC-*.AppImage
    ./PineconeMC-*.AppImage
    ```

    **Flatpak**

    Check the [Releases](https://github.com/ElyPrismLauncher/Launcher/releases/latest) page for a Flatpak bundle if one is provided for your release.

    **Nix / NixOS**

    A `flake.nix` is included in the repository. You can run PineconeMC directly with:

    ```bash
    nix run github:ElyPrismLauncher/Launcher
    ```

    Or add it to your NixOS configuration by pointing your flake input at `github:ElyPrismLauncher/Launcher`.

    **Distro packages**

    Check the [Releases](https://github.com/ElyPrismLauncher/Launcher/releases/latest) page for distribution-specific packages (`.rpm`, `.deb`, or others) that may be included with a given release.
  </Tab>
</Tabs>

## Development builds

Development builds are built automatically from the `develop` branch on every commit. They are available through:

- [GitHub Actions](https://github.com/ElyPrismLauncher/Launcher/actions) — includes builds from contributor pull requests
- [nightly.link](https://nightly.link/ElyPrismLauncher/Launcher/workflows/build/develop) — always points to the latest `develop` build

<Warning>
  Development builds are not intended for everyday use. They may contain bugs or unstable features. Their file sizes are also larger than release builds because debug symbols are included in the binaries.
</Warning>
