---
title: "Get Started with PineconeMC: First Launch to Playing"
description: "Go from a fresh PineconeMC install to playing Minecraft — add your Ely.by or Microsoft account, create an instance, and launch the game."
---

After installing PineconeMC, a setup wizard walks you through the essentials on first launch. This guide covers each step so you can get into a game as quickly as possible.

<Steps>
  <Step title="Launch PineconeMC for the first time">
    Open PineconeMC. The setup wizard starts automatically and guides you through language selection and basic preferences. Complete the wizard and you will land on the main launcher window.
  </Step>
  <Step title="Add your account">
    Click the **Accounts** button in the top-right corner of the launcher, then select **Add Account**.

    - Choose **Ely.by** to log in with an Ely.by account — a browser window opens so you can authenticate on the official Ely.by site.
    - Choose **Microsoft** to log in with a Microsoft account linked to Minecraft Java Edition.
    - Choose **Offline** to play without an account on servers that do not require authentication.

    <Tip>
      Use the Ely.by OAuth2 option rather than an offline Ely.by workaround. OAuth2 means PineconeMC never handles your password — Ely.by issues a limited-privilege token directly to the launcher.
    </Tip>

    <Info>
      For a detailed walkthrough of Ely.by login, see [Ely.by account setup](/accounts/ely-by-login).
    </Info>
  </Step>
  <Step title="Create a Minecraft instance">
    Click **Add Instance** in the toolbar. Select the Minecraft version you want to play. Optionally choose a mod loader:

    - **Fabric** — fast, lightweight, good for performance mods
    - **Forge** — the most widely supported loader for large modpacks
    - **Quilt** — a Fabric fork with additional modding APIs
    - **NeoForge** — a Forge fork with a modernized API

    Click **OK** to create the instance. PineconeMC downloads the selected version automatically.
  </Step>
  <Step title="Configure Java">
    PineconeMC can detect and manage Java for you. Go to **Settings → Java** and click **Auto-detect** to find Java installations on your system. For Minecraft 1.17 and later, Java 17 or newer is required. For older versions, Java 8 works best.

    You can also let PineconeMC download a compatible Java version automatically when you first launch an instance.
  </Step>
  <Step title="Launch your first game">
    Select your instance in the main window and click **Launch**. PineconeMC applies the Ely.by authlib patches automatically, starts the game, and opens the Minecraft log window so you can monitor startup.

    Once the Minecraft title screen appears, you are ready to play.
  </Step>
</Steps>
